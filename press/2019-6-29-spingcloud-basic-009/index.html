# 概念

Docker 是一个开源的应用容器引擎，基于 Go 语言 并遵从Apache2.0协议开源。 Docker 可以让开发者打包他们的应用以及依赖包到一个轻量级、可移植的容器中，然后发布到任何流行的 Linux 机器上，也可以实现虚拟化。 容器是完全使用沙箱机制，相互之间不会有任何接口,更重要的是容器性能开销极低。

docker入门可参考：https://www.kancloud.cn/fymod/docker/783425

本篇环境为macOS系统上安装docker，其他安装可点击这里查看。

# pom.xml插件配置

三个微服务均需添加

	<build>
		<plugins>
			<plugin>
				<groupId>org.springframework.boot</groupId>
				<artifactId>spring-boot-maven-plugin</artifactId>
			</plugin>
			
			<plugin>
                <groupId>com.spotify</groupId>
                <artifactId>docker-maven-plugin</artifactId>
                <version>0.4.13</version>
                <configuration>
                    <imageName>fymod/${project.artifactId}:${project.version}</imageName>
                    <forceTags>true</forceTags>
                    <baseImage>java</baseImage>

                            <targetPath>/</targetPath>
                            <directory>${project.build.directory}</directory>

# docker-compose.yml

  springcloud-9-compose-server: 
    image: fymod/springcloud-9-compose-server:0.0.1-SNAPSHOT
    ports: 
      - 9110:9110
  springcloud-9-compose-client1: 
    image: fymod/springcloud-9-compose-client1:0.0.1-SNAPSHOT
